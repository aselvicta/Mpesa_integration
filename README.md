# Mpesa_tz

A lightweight and developer-friendly Python package for integrating M-Pesa Tanzania STK Push payments into Python applications.

Built for developers using:
- Django
- Flask
- FastAPI
- Pure Python projects

---

## Features

- STK Push payment requests
- OAuth token generation
- Simple and clean API
- Easy integration with backend systems
- Sandbox and production support
- Designed for reusable package distribution

---

## Installation

```bash
pip install mpesa-tz
